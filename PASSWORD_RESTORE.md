# Восстановление пароля VPS

## Проблема

Пароль был автоматически изменен для безопасности, но это было сделано по ошибке.

## Решение: Восстановить старый пароль

### Через консоль VPS провайдера

1. Зайдите в панель управления вашего VPS провайдера
2. Откройте консоль (VNC/Console)
3. Войдите как root (если требуется)
4. Выполните команду:

```bash
echo 'root:Ty8c5FGqLG695-i9j5V%' | chpasswd
echo "✅ Пароль восстановлен!"
```

5. Проверьте:
```bash
ssh root@144.124.225.15
# Введите пароль: Ty8c5FGqLG695-i9j5V%
```

## После восстановления

Приложение продолжает работать нормально:
- Docker контейнер запущен
- Доступен на http://144.124.225.15:3001

Осталось только:
1. Настроить Nginx (см. QUICK_START.md)
2. Настроить SSL сертификат
