# Claude Code Instructions

## Deployment

After any code changes, always deploy:
1. Commit changes to git
2. Push to `origin main`
3. CI/CD (GitHub Actions) will automatically build and deploy to VPS

Deploy is automatic on push to main — no manual deploy script needed.

## Design Reference

`design-example.html` — эталонный пример того, как должен выглядеть дашборд. Используй его как визуальный ориентир при любых изменениях UI.
